Project 2

Meat Rush: 3D Endless Runner Game

Game Engine: Unity
Language Used: C#
Platform: PC

Game Summary

"Meat Rush" is a 3D endless runner featuring a cartoony fox in a vibrant, low-poly world. Players guide the fox to jump and roll, avoiding obstacles like boxes, blades, and trees. The game spans three themed locations: desert, water, and tropical forest. Collect meats to boost your score while dodging challenges!

Project 3

FrostFire Legends: 3D Open World Adventure

Game Engine: Unity
Language Used: C#
Platform: PC

Game Summary

FrostFire Legends is a 3D open-world adventure survival game set in the medieval and snow-covered landscape. Players control Paragon Kwang, a warrior wielding fire powers through his sword, battling mythical enemies like ice golems and rhinos. The game features dynamic snowfall, high-res textures, and optimized performance with enemy pooling and occlusion culling.
